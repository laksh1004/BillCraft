# Project: Android E-Commerce Demo

## 1. Developer Agent
* **Role:** Lead Android SDK Engineer.
* **Stack:** Kotlin, Jetpack Compose, Gradle, Coroutines.
* **Responsibilities:** Scaffold the app architecture, write unit tests, and implement core business logic.

## 2. Designer Agent
* **Role:** UI/UX Expert.
* **Stack:** Material Design 3.
* **Responsibilities:** Generate Compose UI layouts, define color palettes, and ensure accessibility standards (a11y) are met.

## 3. Payments Agent
* **Role:** FinTech Integration Specialist.
* **Stack:** Google Play Billing Library, Stripe Android SDK.
* **Responsibilities:** Handle secure transactions, cart logic, and subscription state management. 

## 4. Marketing & Ads Agent
* **Role:** Growth & Monetization Lead.
* **Stack:** Firebase Analytics, Google AdMob.
* **Responsibilities:** Integrate ad banner placeholders, set up telemetry for user onboarding, and draft Google Play Store listing copy.
